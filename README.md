## 搬瓦工重新买，先搞清楚自己的情况

搬瓦工重新买分两种完全不同的情况，操作路径也不一样：

**情况一：老 VPS 到期被删，但账号还在**

这是最常见的。VPS 到期后，搬瓦工有一段宽限期（账单生成后 7 天），超过这个时间不续费，VPS 会被彻底删除，数据没了，服务没了，但你的**账号本身还在**。

这种情况直接用老账号重新登录，进去下单就行。不用注册新账号，也不要注册——一个邮箱对应一个账号，搬瓦工不允许同一个人名下开多个账号。

**情况二：账号密码都不记得了**

去搬瓦工官网的登录页，点 "Forgot Password"，输入注册时的邮箱，它会发重置链接过来。找不到原始邮箱的话，只能往后翻一翻当年的付款记录，支付宝或者 PayPal 的账单里通常能找到线索。

找回账号之后，流程和情况一完全一样，重新进去下单就好。

**情况三：彻底换邮箱或者从零开始**

真的是新人，或者原来那个邮箱已经废掉了找不回来，才注册新账号。注册时候有几个坑要提前说：

- 国家一定选 **China**，不然大概率被判欺诈，订单直接取消
- 不要挂代理软件下单，同理，容易触发反欺诈
- 姓名可以不用真名，但不能乱填，至少得像个人名（比如 Zhang Wei 这种），填 "asdfgh" 这种是直接废单的
- 邮箱必须真实，这是你登录账户和找回密码的唯一依据

**第五步：选支付方式，付款**

支持支付宝（推荐）、微信支付、PayPal、信用卡，中国用户用支付宝最顺手，扫码就完事了。
